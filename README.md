# Pulsar Star Prediction using Machine Learning

## Overview
This project i made in my 3rd year for my Minor Project when i was very new to Machine Learning Concepts.
The project predicts whether a celestial object is a Pulsar Star or not using Machine Learning classification algorithms.

## Algorithms Used
- Support Vector Machine (SVM)
- Naive Bayes
- Decision Tree

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Features
- Data preprocessing
- Feature scaling
- Model training
- Accuracy comparison
- Interactive prediction interface

## Dataset
Dataset taken from Kaggle.

## Results
Compared accuracy of multiple classification models and analyzed performance.

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
